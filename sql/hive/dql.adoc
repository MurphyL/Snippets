## 分组取TopN

[source, sql]
.group_top_n.sql
----
SELECT *
  FROM (SELECT label,
               row_number () OVER (PARTITION BY label ORDER BY num DESC) rn,
               num
          FROM test.group_top_n) a
 WHERE a.rn <= 3
----
