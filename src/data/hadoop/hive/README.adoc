---
title: Hive
---

`Hive`是`Facebook`于2008年开源的数据仓库框架。`Hive`使用类似于SQL的查询语言——`HQL`，能够将用户编写的`QL`转化为相应的`MapReduce`程序基于`Hadoop`执行。

`Hive`适合用来对一段时间内的数据进行分析查询。Hive 不应该用来进行实时的查询（`Hive`的设计目的，也不是支持实时的查询），因为它需要很长时间才可以返回结果。
